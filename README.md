# 微信跳一跳辅助工具

### 依赖

- adb
- 安卓手机

### 使用方法

在bin目录中直接下载pc的对应版本，mac和linux请给与可执行权限

在命令行直接运行可执行程序 `./bargo-xxx -s 1.325`

s参数为跳跃速度系数，需要根据你的手机分辨率做调整，我的手机是1080*2220 用这个默认数值刚好合适
